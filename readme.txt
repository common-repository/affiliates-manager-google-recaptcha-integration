An addon for the Affiliates Manager plugin to add Google reCAPTCHA to the registration page

== Description ==

This addon allows you to specify your Google reCAPTCHA V2 API credentials in the addon settings. 

When affiliates go to the registration page, they will need to manually verify that they are not a robot before submitting the form.

This addon requires the [Affiliates Manager Plugin](https://wordpress.org/plugins/affiliates-manager/).

After you install this addon, go to the Google reCAPTCHA settings interface, specify your V2 API credentials and save the settings.

Read the following page for step by step usage documentation:
https://wpaffiliatemanager.com/affiliates-manager-google-recaptcha-integration/

== Installation ==

1. Upload `affiliatemgr-google-recaptcha-addon` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
